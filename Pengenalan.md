## Apa itu Angular?
Angular adalah JavaScript Framework untuk membuat client-side application menggunakan HTML, CSS dan JavaScript.

## Why Angular?
- Expressive HTML
- Powerful Data Binding
- Modular By Design
- Built-in Back-End Integration
- Built for Speed
- Modern
- Simplified API
- Enhances Productivity
